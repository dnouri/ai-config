# ai-config

Personal configuration for [pi coding agent](https://buildwithpi.ai).

## Contents

- **AGENTS.md** - TDD workflow with Tim Peters & Kent Beck pair programming
- **skills/tmux** - Run long-running processes in tmux sessions
- **extensions/notify.ts** - Desktop notification when agent finishes

## Install

```bash
pi install dnopi
```

Or symlink manually from `~/.pi/agent/`.
